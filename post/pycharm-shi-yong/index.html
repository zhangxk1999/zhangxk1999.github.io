Pycharm新建项目，new environment 和 existing interpreter的区别

New Environment部分是选择新建项目所依赖的环境[便于管理不同项目]
base interpreter 理解为在这之上进行添加新的包等操作

existing interpreter关联本地已经存在的解释器[可以减少占用，并且不需要再次去下载包]
